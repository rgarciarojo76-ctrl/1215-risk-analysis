# Análisis de Riesgos (Imágenes)

Este proyecto es una herramienta de la **Dirección Técnica IA LAB** diseñada para agilizar la identificación de riesgos mediante visión artificial.

## Funcionalidad Principal
- **Análisis Visual**: Procesamiento de imágenes de entornos laborales para detectar factores de riesgo.
- **Simulación**: Generación de imágenes "después" con medidas correctivas aplicadas.
- **Informes**: Generación de evidencias documentales.

## Configuración
El nombre interno del proyecto es `direccion-tecnica-risk-analysis`.
