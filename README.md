# Guía de Instalación y Ejecución

## Prerrequisitos
Necesitas tener **Node.js** instalado en tu Mac.
Si no lo tienes, descárgalo e instálalo desde: [https://nodejs.org/](https://nodejs.org/)

## Pasos para ejecutar la aplicación

1. **Abrir la Terminal**
   - Pulsa `Comando (⌘) + Espacio`.
   - Escribe "Terminal" y pulsa Enter.

2. **Navegar a la carpeta del proyecto**
   Copia y pega este comando en la terminal:
   ```bash
   cd /Users/rubengarciarojo/.gemini/antigravity/playground/primal-omega
   ```

3. **Instalar las dependencias**
   Copia y pega:
   ```bash
   npm install
   ```
   *Si aparece un error diciendo "command not found", necesitas instalar Node.js primero.*

4. **Arrancar la aplicación**
   Copia y pega:
   ```bash
   npm run dev
   ```

5. **Abrir en el navegador**
   Verás un link como `http://localhost:5173/`. Mantén pulsada la tecla `Comando` y haz clic en el link para abrirlo.
